# 结构代码如何书写？



注意：其控制器基础结构代码，不需要自己去手动编写，可以通过artisan命令行来自动生成。

因此需要记住对应的命令：

```shell
#php artisan make:controller 控制器名（大驼峰）Controller关键词
```

例如：使用artisan命令创建TestController.php文件。先确定命令：

```shell
#php artisan make:controller TestController
```
